## 系统模块总览

MFG系统是CVTE的制造执行系统，涵盖以下核心模块：

| 模块 | 说明 |
|------|------|
| **MES (生产管理)** | 生产制造执行管理 |
| **WMS (仓储管理)** | 仓库入库、库内、出库管理 |
| **QMS (质量管理)** | 质量策划、IQC/IPQC/OQC、制程质量、客户质量、市场质量 |
| **SOP (功能手册)** | 各模块操作手册 |
| **FAQ** | 常见问题解答 |
| **场景解决方案** | 业务场景解决方案 |
| **OA** | 办公自动化(知识中心等) |

## 质量管理核心流程

基于文档结构可识别的质量管理核心流程：

1. **质量策划** - 前置质量规划
2. **客户质量管理** - 售后维修费用转嫁(客责)
3. **研发设计质量** - 研发阶段质量管控
4. **物料质量管理**
   - 供应商质量管理(供方技术交流、物料参数、测试数据导入、物料预警)
   - 物料检验(进料检验SPC分析)
5. **制程质量管理** - 生产异常处理、制程稽核
6. **质量保证** - 质量责任处理闭环(处理单、保证金、暂停支付、赠品抵扣)
7. **市场质量管理** - EWP分析(研发/生产/市场/客户)、整改组分析
8. **问题管理** - 问题跟踪管理
