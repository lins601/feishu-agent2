# 表单设计

# 表单设计页功能介绍


## 页面概述


表单设计页（路由：`/workspace/{appId}/feature/formDesign`）是天舟云低代码平台的可视化表单设计器。通过拖拽式操作，用户可以快速构建各类业务表单，支持复杂的布局嵌套、字段配置和校验规则设置。


## 页面结构


表单设计页采用经典四区域布局：


```
┌─────────────────────────────────────────────────────────┐
│                     顶部工具栏                            │
├──────────┬──────────────────────────────┬───────────────┤
│          │                              │               │
│ 左侧组件  │        中间设计画布            │  右侧配置面板  │
│ 库面板    │        + 工具栏               │               │
│          │                              │               │
└──────────┴──────────────────────────────┴───────────────┘
```

## 功能模块详解


### 1. 顶部工具栏（formHeaderBar）


顶部工具栏分为左侧面包屑导航和右侧功能按钮区。


#### 1.1 左侧面包屑导航


显示当前表单的层级结构：


```
分类名称 > 设计方案名称
```


#### 1.2 右侧功能按钮


| 按钮 | 功能说明 | 操作说明 |
| --- | --- | --- |
| **帮助** | 跳转到帮助文档 | 打开天舟云官方文档页面 |
| **预览** | 打开预览抽屉 | 在抽屉中展示表单的实际渲染效果，支持数据填写测试 |
| **保存** | 保存表单设计 | 执行校验后保存，包含：必填项校验 + 组件编码唯一性校验 |


#### 1.3 设计方案切换


点击顶部「查看方案」按钮，显示 Popover 弹窗：


- **方案列表**：显示当前表单的所有设计方案
- **快速切换**：点击方案名称即可在新标签页打开该方案
- **查看更多**：打开设计方案列表抽屉，进行完整的管理操作

### 2. 左侧组件库面板（formEle）


左侧组件库面板提供可拖拽到画布的表单组件，通过 Tab 切换不同的组件类别。


#### 2.1 顶部 Tab 导航


| Tab 名称 | 说明 |
| --- | --- |
| **表单元素** | 可拖拽的基础组件、容器组件、布局组件、高级组件 |
| **属性元素** | 对象属性配置（与表单数据模型关联） |
| **模板库** | 预制表单模板中心 |


#### 2.2 Extra 功能区


在 Tab 栏右侧的额外功能区：


- **全局样式配置**：点击打开底部抽屉，配置表单的全局统一样式


#### 2.3 组件分类及列表


##### **基础组件**


| 组件名称 | 组件 Key | 说明 |
| --- | --- | --- |
| 分割线 | `DIVIDER` | 表单区域分割线，用于视觉分隔 |


##### **容器组件**


| 组件名称 | 组件 Key | 说明 |
| --- | --- | --- |
| tab容器 | `TAB` | Tab 标签页容器，支持多个 Tab 页签 |
| card容器 | `CARD` | 卡片容器，用于分组展示表单字段 |
| 懒加载列表 | `LAZY_LOAD_LIST` | 懒加载列表容器，支持滚动加载 |
| 扩展表 | `BASIC_INFO_EXT_GROUP` | 扩展表容器 |
| 明细表 | `TABLE` | 明细表容器，支持多行数据录入 |
| 明细表card | `TABLE_FORM` | 明细表卡片样式容器 |
| 按钮容器 | `BTN_BLOCK` | 按钮组容器 |
| 表格容器 | `TABLE_CONTAINER` | 表格容器 |


##### **布局容器**


| 组件名称 | 组件 Key | 说明 |
| --- | --- | --- |
| 栅格容器 | `GRID_CONTAINER` | 栅格布局容器，支持多种栅格比例 |
| 字段布局 | `FLEX_CONTAINER` | Flex 弹性布局容器 |


##### **高级组件**


| 组件名称 | 组件 Key | 说明 |
| --- | --- | --- |
| 人员单选 | `PERSON` | 人员选择器（单选） |
| 人员多选 | `MULTI_PERSON` | 人员选择器（多选） |
| 组织单选 | `ORGANIZTION` | 组织架构选择器（单选） |
| 组织多选 | `MULTI_ORGANIZTION` | 组织架构选择器（多选） |
| 上传图片 | `MULTI_PICTURES` | 图片上传组件 |


##### **自定义组件**


- 由第三方动态注入的自定义组件
- 可通过插件机制扩展

### 3. 中间设计画布工具栏（formMakeToolBar）


设计画布顶部的工具栏，提供画布操作和设备预览功能。


#### 3.1 左侧设备切换区


**设备类型切换**：


- **PC**：PC 端视图（默认）
- **移动端**：移动端视图


**移动端设备尺寸选择**（仅在移动端模式下可用）：


- 支持多种移动设备尺寸预设
- 自定义设备尺寸


#### 3.2 右侧功能按钮区


| 按钮 | 功能说明 | 操作说明 |
| --- | --- | --- |
| **清空表单** | 清空所有设计内容 | Popover 二次确认后清空画布上的所有组件 |
| **版本管理** | 历史版本管理 | 打开版本列表，查看、启用、预览历史版本 |
| **AI一键美化** | AI 自动优化布局 | 调用 AI 服务自动优化表单布局和样式 |

### 4. 中间设计画布（formMake）


中间画布区域是表单设计的主工作区，支持拖拽放置和可视化编辑。


#### 4.1 画布功能


- **拖拽放置**：从左侧组件库拖拽组件到画布
- **点击选中**：点击组件后，右侧面板显示配置项
- **嵌套布局**：容器组件可放置子组件，支持多层嵌套
- **按钮组管理**：配置表单按钮的显示/隐藏、位置、对齐方式
- **空状态引导**：无组件时显示空状态图片，引导用户操作


#### 4.2 布局模板


##### 栅格布局预设


| 布局名称 | 栅格比例 |
| --- | --- |
| 单列 | 24 |
| 1:1 | 12:12 |
| 1:2 | 8:16 |
| 2:1 | 16:8 |
| 1:1:1 | 8:8:8 |
| 1:4:1 | 4:16:4 |
| 四列 | 6:6:6:6 |
| 六列 | 4:4:4:4:4:4 |


##### Flex 布局预设


- 1 列布局
- 2 列布局
- 3 列布局
- 4 列布局
- 5 列布局


#### 4.3 按钮组配置


表单支持配置按钮组，可选择：


- **位置**：顶部 / 底部
- **对齐方式**：左对齐 / 居中 / 右对齐
- **显示/隐藏**：控制按钮组是否显示
- **新增按钮**：点击「新增按钮」添加自定义按钮

### 5. 右侧属性配置面板（formConf）


右侧配置面板根据选中组件显示不同的配置项，通过三个独立的配置区域组织。


#### 5.1 控件信息（基础配置）


**配置项说明**（根据组件类型动态显示）：


| 配置项 | 字段类型 | 说明 |
| --- | --- | --- |
| 属性类型 | 下拉选择 | 关联的数据对象属性 |
| 标题名称 | 输入框 | 组件显示的标签文本 |
| 组件编码 | 输入框 | API 名称，用于接口调用，全局唯一 |
| 默认值 | 多种 | 根据组件类型显示对应的默认值配置 |
| 是否显示 | 开关 | 控制组件是否在表单中显示 |
| 是否可编辑 | 开关 | 控制组件是否可编辑 |
| 是否启用 | 开关 | 控制组件是否启用 |
| 标题提示 | 输入框 | 标题旁的提示信息（Tooltip） |
| 框内提示 | 输入框 | 输入框内的占位文本（Placeholder） |


**高级配置**：
| 配置项 | 说明 |
|——–|——|
| 组件宽度 | 设置组件宽度（百分比或像素） |
| 提交方式 | 配置数据提交方式 |
| 虚拟字段 | 标记为虚拟字段（不提交到后端） |
| 自定义样式 | 自定义 CSS 样式 |


#### 5.2 校验规则


支持的校验类型：


| 校验类型 | 适用组件 | 配置项 |
| --- | --- | --- |
| **必填** | 所有输入组件 | 必填校验开关 + 错误提示信息 |
| **最大长度** | 文本类组件 | 最大字符数 |
| **最小长度** | 文本类组件 | 最小字符数 |
| **最大值** | 数字类组件 | 最大数值 |
| **最小值** | 数字类组件 | 最小数值 |
| **正则匹配** | 文本类组件 | 正则表达式 + 错误提示 |
| **特殊类型** | 多种 | 邮箱、手机号、身份证等预设校验 |
| **数字比较** | 数字类组件 | 与其他数字字段的比较校验 |
| **文件体积** | 文件上传类 | 单文件最大体积（MB） |
| **文件数量** | 文件上传类 | 最大上传数量 |
| **文件类型** | 文件上传类 | 允许的文件类型（扩展名） |


#### 5.3 外观配置


**通用外观配置**：


| 配置项 | 说明 |
| --- | --- |
| 标签对齐方式 | 左对齐 / 居中 / 右对齐 |
| 标签列占比 | labelCol（栅格数 0-24） |
| 内容列占比 | wrapperCol（栅格数 0-24） |


**组件专属外观配置**（根据组件类型显示）：


- 文本类组件：输入框大小、前缀图标、后缀图标
- 选择类组件：下拉面板高度、选项展示方式
- 日期类组件：日期格式、时间选择器配置
- 等等…

## 操作流程


### 新建表单流程


- 从左侧组件库拖拽布局容器到画布
- 在布局容器中拖拽需要的表单组件
- 点击组件，在右侧面板配置基础信息、校验规则、外观样式
- 点击顶部「保存」按钮保存设计
- 点击「预览」按钮测试表单效果


### 组件嵌套流程


- 拖拽容器组件（如 Tab、Card）到画布
- 在容器内部拖拽子组件
- 支持多层嵌套（如：栅格容器 > Card > 明细表 > 字段）

## 常见问题


### Q: 如何添加新的表单字段？


A: 从左侧「表单元素」Tab 拖拽需要的组件到画布，点击组件后在右侧「控件信息」面板配置字段属性。


### Q: 如何实现多列表单布局？


A: 使用「栅格容器」或「字段布局」组件，选择预设的列布局模板（如 1:1、1:2、1:1:1 等），然后在各个栅格中放置字段。


### Q: 如何设置字段为必填？


A: 点击目标组件，在右侧「校验规则」面板中开启「必填」开关，并填写错误提示信息。


### Q: 如何实现字段之间的联动？


A: 字段联动需要通过「属性联动」功能实现，在表单属性设置中配置字段联动规则。


### Q: 如何预览表单效果？


A: 点击顶部「预览」按钮，打开预览抽屉，可以在真实渲染环境中测试表单填写和校验。


### Q: 如何恢复之前的设计版本？


A: 点击「版本管理」按钮，查看历史版本列表，点击「预览」查看版本内容，点击「启用」恢复到该版本。


### Q: 保存时提示”组件编码重复”怎么办？


A: 组件编码（API名称）必须全局唯一。检查右侧「控件信息」面板中的「组件编码」字段，修改重复的编码后重新保存。


### Q: 如何使用 AI 一键美化功能？


A: 点击画布工具栏的「AI一键美化」按钮，系统会自动调用 AI 服务优化表单布局和样式。
