## 1. 模块定位与入口


**高级管理**用于在**应用/表单（含流程表单）**维度配置「运行与治理」类能力：数据**唯一性**、**列表/表单权限**（与流程字段权限衔接）、**导入/导出**模板与字段策略、**数据标题**拼装规则等。
与 **表单设计**（界面）、**流程设计**（节点权限）、**数据模型**（字段定义）互补：高级管理更偏**整单治理与数据生命周期外围规则**。

**入口**：


- 顶栏选中 **「高级管理」**（与 **表单设计**、**数据列表**、**流程设计**、**属性联动**、**表单转换** 等并列，是否展示以授权为准）。
- 路由（代码配置）：`/workspace/{appId}/feature/advanceManager`（常带 `id`、`featureType` 等，以环境为准）。

**AI 检索口诀**：


- 「重复提交/唯一性」→ **唯一校验**

- 「Excel 导入模板、必填、示例列」→ **导入设置**
- 「导出带哪些列」→ **导出设置**
- 「列表/待办上那条标题怎么来的」→ **数据标题**

## 3. 功能速查（关键词 → 路径 → 是否支持）

| --- | --- | --- |

| 启用/停用唯一规则 | 规则列表 **启用状态**；弹窗 **启用规则** | **支持** |
| 按属性组配唯一 | **新增规则** → **属性组** + **需校验的属性** | **支持** |
| 创建时谁能操作 | **权限管理** Tab **创建态** | **支持** |
| 已有数据谁可操作 | **权限管理** Tab **非创建态** | **支持** |
| 新建权限组 | **新增权限组** | **支持** |
| 权限组搜索 | 右上角按 **权限名称** 搜索 | **支持** |
| 成员：全员或指定人 | **权限成员**：**全部用户** / **自定义**（**设置成员**） | **支持** |
| 字段可见可编是否跟流程一致 | **字段权限** 文案：**继承流程设计中发起节点的字段权限设置** | **支持**（策略描述以界面为准） |
| 列表/详情按钮可用性 | **按钮权限** 勾选（复制、删除、创建、编辑、复制创建、保存、跳转等） | **支持** |
| 按数据范围控权 | **条件配置**：**全部数据** / **自定义** + **新增条件** | **支持** |
| 条件 AND/OR、属性比较 | **新增条件** → **条件图谱** | **支持** |
| 复制/删除权限组 | 列表 **复制**、**删除** | **支持** |
| 导入模板说明 | **导入设置** **模板说明**（如 ≤1000 字） | **支持** |
| 导入列必填、示例 | 表格 **必填**、**示例** | **支持** |
| 细目表共用主表列配置 | **是否应用于细目表** | **支持** |
| 多 Sheet 导入 | **是否显示在额外 sheet**、**额外显示字段** | **支持**（部分行可置灰） |
| 保存导入配置 | **导入设置** **保存** | **支持** |
| 导出勾选字段 | **导出设置** 主表字段复选框 | **支持** |
| 拼接标题（变量+文本） | **数据标题** **标题设置** + **可用变量** | **支持** |
| 保存标题配置 | **数据标题** **保存** | **支持** |

## 4. 唯一校验


用于约束：指定 **属性组** 内一个或多个 **属性** 的组合在业务范围内 **不可重复**（具体校验时机与范围以后端为准）。

### 4.1 列表


| 列（示例） | 含义 |
| --- | --- |
| **操作** | 编辑、删除等 |
| **属性组** | 规则所属分组 |
| **校验属性** | 参与唯一性判断的字段 |
| **启用状态** | 是否生效 |
| **更新人 / 更新时间** | 审计 |


**工具按钮**：**新增规则**（绿）、**刷新列表**（蓝）。

### 4.2 新增规则弹窗


| 字段 | 说明 |
| --- | --- |
| **属性组** | 必填；下拉 |
| **启用规则** | 开关（截图示例默认为「否」） |
| **请选择需校验的属性** | 必填；多选需校验字段 |


**取消 / 确认** 关闭或保存。

---


## 5. 权限管理


区分单据生命周期的两段：**创建态**（新建单据过程中）与 **非创建态**（已存在实例后的查看/办理等），分别维护 **权限组**。

### 5.1 公共能力


- **新增权限组**
- **搜索**：按 **权限名称** 筛选
- **列表字段**（示例）：**权限名称**、**权限描述**、**权限成员**、**字段权限**、**按钮权限**
- **行操作**：**编辑**、**复制**、**删除**
- **分页**：总数与翻页


### 5.2 权限组表单（新建/编辑）


| 区块 | 说明 |
| --- | --- |
| **名称** | 必填 |
| **描述** | 多行说明 |
| **权限成员** | **全部用户** 或 **自定义**；自定义时 **设置成员**（显示已选条数） |
| **条件配置**（非创建态等场景可见） | **全部数据** 或 **自定义**；自定义时配合 **新增条件** |
| **字段权限** | 常见说明：继承 **流程设计** 中 **发起节点** 的字段权限；或可选 **自定义** 再 **设置**（以界面为准） |
| **按钮权限** | 多选：**复制**、**删除**、**创建**、**编辑**、**复制创建**、**保存**、**跳转** 等 |


### 5.3 新增条件弹窗（数据范围）


用于 **自定义** 条件配置时，定义「哪些数据」命中该权限组。


| 区块 | 说明 |
| --- | --- |
| **标题** | 必填 |
| **编码** | 系统自动生成，只读（如 `CONDITION_...`） |
| **分类** | 必填；下拉（与当前业务分类绑定） |
| **描述** | 备注 |
| **条件图谱** | 逻辑：**且**（AND，可扩展为或，以产品为准）；多行：**属性**、**运算符**、**值类型**（如 **值**）、**取值**；支持 **+** 增行、删除 |


**取消 / 确定** 保存条件。

## 6. 导入设置


配置 **批量导入** 模板行为：哪些字段出现在模板中、是否必填、示例数据、是否与明细表联动、是否额外 Sheet 等。

### 6.1 顶区


- **保存**：提交当前导入配置
- **模板说明**：多行文本（示例上限 **1000** 字），用于告知使用模板注意事项


### 6.2 配置表（列说明）


| 列 | 含义 |
| --- | --- |
| **表名** | 如 **主表单**；可含层级（子表以环境为准） |
| **字段名** | 导入列对应的业务字段 |
| **必填** | 导入时是否要求该列有值 |
| **是否应用于细目表** | 同一规则是否延伸到明细表 |
| **是否显示在额外 sheet** | 是否拆到附加 Sheet（部分字段可能禁用该选项） |
| **额外显示字段** | 附加展示/映射用字段名 |
| **示例** | 模板中的示例单元格内容，占位 **请输入示例数据** |


左侧常有行 **勾选**，用于批量选择（具体批量操作以产品为准）。

---


## 7. 导出设置


通过 **复选框** 选择导出文件中 **包含的主表字段**（及部分系统/流程字段）。

**示例字段（截图）**：主键、编号、数据版本、流程标签、分类、生命周期阶段、创建用户、创建时间、更新用户、更新时间、**工作流关联模板 ID**、**工作流实例 ID** 等。

表头常带 **全选**，便于一次性勾选/取消。

---


## 8. 数据标题


用于配置单据在列表、关联展示等场景的 **动态标题**：由 **固定文案** 与 **变量（字段值）** 拼接而成。

### 8.1 标题设置


- 交互区形如：**标题设置 =** 后接 **可编排内容**（片段中可插入蓝色标签变量，如 **主键**）。
- 示例语义：`主键` 的值 + 静态「你的标题」→ 展示为连续标题。


具体拼接规则（分隔符、空值处理）以产品实现为准。

### 8.2 可用变量


- **搜索变量**：检索可插入字段
- 树下按 **主表单** 列出字段，展示 **字段名** 与 **类型**（单行文本、流水号、数字、多行文本、树形单选、下拉单选、人员单选等），点击即可加入标题表达式


### 8.3 保存


**保存** 持久化标题配置。

---


## 9. 模块协作关系（给 AI 的固定话术）


- **流程设计** 配置 **节点级** 字段权限；**高级管理 · 权限管理** 中 **字段权限** 常与 **发起节点** 策略 **继承** 关系一起出现——应答时提醒用户两端不要冲突，以实际「继承/自定义」选项为准。
- **数据列表 · 导出设置**（若独立菜单）与 **高级管理 · 导出设置** 可能并存或分工，用户问「导出」时需看其当前在哪个 Tab；本文专指 **高级管理** 内导出。
- **唯一校验** 管 **数据完整性**；**权限管理** 管 **谁能看/改/点按钮**；**导入/导出设置** 管 **文件结构**；**数据标题** 管 **展示聚合文案**。
- **新增条件** 的条件图谱与 **属性联动**、**流程条件** 概念相近但入口不同，勿混用路径。
