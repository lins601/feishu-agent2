# 条件分支节点

# 审批节点


### 功能介绍


条件分支节点可以将一个流程设计分成多个分支，每个分支可设置自己的流转条件,当流转到该节点时将找到满足条件的分支进行流转


### 使用场景


在审批流程中，如果需要根据不同判断条件设置不同的审批人时，我们就可以通过设置审批条件，在一个流程中设置多个流程分支。


```
例如：员工在请假时，请假时长小于 7 天时，给主管审批，请假时长大于 7 天时，给总经理审批
```
