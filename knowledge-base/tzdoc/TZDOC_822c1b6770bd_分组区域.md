# 分组区域

# 1. 简介


### 1.1 功能说明


用于分组展示当前表单数据的过滤条件，可以快速筛选查看数据。提高数据管理效率和用户体验。


### 1.2 支持模式


| 模式 | 说明 | 应用场景 |
| --- | --- | --- |
| 分类 | 展示当前表单的分类树，查看每个分类数据情况 | 一般是对象和变更这种多级分类表单 |
| 表单属性 | 通过属性构建数据树，每一层对应一个属性，上下数据级联过滤，可以快速查看某一维度有哪些数据，以及当前维度下另一个维度有哪些数据 | 业务数据查看有层级例如：查看某一事业部某一产品型号 |
| 关联表单数据 | 以关联的表单数据作为过滤条件，支持树形和列表方式 | 左树右列表 |
| 连接器 | 以外部数据作为过滤条件，支持树形和列表方式 | 过滤数据来源外部/数据库其他表 ，左树右列表外部数据方式 |
