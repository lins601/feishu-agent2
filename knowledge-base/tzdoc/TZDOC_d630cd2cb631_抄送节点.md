# 抄送节点


### 功能介绍


流程支持用户在设置流程时加入抄送节点，用于在审批人审批后给抄送人发送消息提醒，抄送人不需要审批和执行


### 使用场景


员工发起报销审批的时候，财务人员审批同意了，那么领导需要查看这部分报销是在哪里，就可以设置领导为抄送人，到时候领导就可以直接收到这个消息通知。。


### 操作步骤


##### 一、设置抄送人


```
抄送人设置和审批节点的审批人设置一致
```
