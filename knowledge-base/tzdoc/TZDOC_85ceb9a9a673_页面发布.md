## 1. 模块定位与入口


**页面发布**用于把当前表单/流程应用配置为可访问的**前台页面资源**：指定挂到哪个**上级菜单**、**发布类型**（是否含列表页、详情页）、以及列表/详情各自的 **编码、名称、访问地址、路由参数** 等，并可选择是否**自动创建页面资源**（菜单/权限等系统侧对象，语义以产品为准）。

**入口**：


- 顶栏选中 **「页面发布」**（与 **表单设计**、**数据列表**、**流程设计**、**属性联动**、**表单转换** 等并列；部分环境 Tab 在「更多」折叠中，以实际为准）。
- 右上角 **保存** 提交配置。

## 3. 功能速查（关键词 → 路径 → 是否支持）


| 用户可能问的关键词 | 配置路径 | 是否支持（据截图） |
| --- | --- | --- |
| 挂到哪个菜单下 | **页面发布信息** → **上级页面** | **支持** |
| 只要列表 / 只要详情 / 列表+详情 | **发布类型** | **支持**（截图示例：`列表+详情`） |
| 是否自动生成菜单等资源 | **创建页面资源** 开关 | **支持** |
| 列表页路由、资源名 | **列表配置** | **支持** |
| 详情页单独路由 | **详情配置** | **支持** |
| 保存所有发布项 | 顶栏 **保存** | **支持** |
| 自定义前端包路径 | **动态组件路径** | **支持** |
| URL 参数 JSON | **路由参数** 多行文本 | **支持** |

## 4. 页面发布信息


主配置区标题多为 **「页面发布信息」**，包含：


| 字段 | 必填 | 说明 |
| --- | --- | --- |
| **上级页面** | * | 下拉；选择父级菜单/页面节点，决定发布页在应用导航中的**层级位置**。截图示例：`账户配置`。 |
| **发布类型** | * | 下拉；决定下方是否出现 **列表配置**、**详情配置** 或两者。截图示例：**列表+详情**。 |
| **创建页面资源** | — | 开关；**开**：通常自动创建或同步系统内**页面资源**（如权限、资源注册等）；**关**：不自动创建（截图示例曾为「否」）。具体行为以产品说明为准。 |


---


## 5. 列表配置


当 **发布类型** 含 **列表** 时出现。用于定义**列表页**在系统中的技术标识与访问方式。


| 字段 | 必填 | 说明 |
| --- | --- | --- |
| **页面编码** | * | 列表页唯一编码。 |
| **页面名称** | * | 列表页显示名称。 |
| **资源分类** | * | 资源归类（下拉）。 |
| **权限类型** | — | 访问控制策略（下拉）。 |
| **页签展示方式** | — | Tab 展示样式（下拉）。 |
| **标签标志** | — | 标签/标记用途的文本。 |
| **页面图标** | — | 图标类名或资源标识。 |
| **动态组件路径** | — | 自定义前端组件加载路径（协议/路径以平台约定为准）。 |
| **访问地址** | * | 列表页 **URL 路径**（用户或网关访问用）。 |
| **路由参数** | * | 多行；**路由/查询参数**定义（常见为 JSON 或平台约定格式），需与运行时路由解析一致。 |


---


## 6. 详情配置


当 **发布类型** 含 **详情** 时出现。**字段集合与「列表配置」平行**，用于 **详情/表单页** 独立一套 **页面编码、名称、资源分类、访问地址、路由参数** 等，避免列表与详情共用同一资源定义。


| 字段 | 必填 | 说明 |
| --- | --- | --- |
| **页面编码** | * | 详情页编码（独立于列表）。 |
| **页面名称** | * | 详情页显示名称。 |
| **资源分类** | * | 同列表侧语义。 |
| **权限类型** | — | 同列表侧语义。 |
| **页签展示方式** | — | 同列表侧语义。 |
| **标签标志** | — | 同列表侧语义。 |
| **页面图标** | — | 同列表侧语义。 |
| **动态组件路径** | — | 同列表侧语义。 |
| **访问地址** | * | 详情页访问路径。 |
| **路由参数** | * | 详情页路由参数定义。 |


---


## 7. 发布类型与区块显隐（给 AI 的固定话术）


- **列表+详情**：同时展示 **列表配置** 与 **详情配置**，用户典型路径为「列表点进详情」，两套 **访问地址 / 路由参数** 分别配置。
- 若仅有 **列表** 或仅有 **详情**（选项以线上下拉为准），则只展示对应一块配置区。
- 所有标 **红色 *** 的项保存前一般需填写完整，否则易发布失败或路由无法注册。
