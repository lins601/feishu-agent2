# OA原生流程下推

#### 接口说明


 接口名称：OA原生流程下推
 接口地址：https://域名/cvte/api/ekpKmReview.do
 请求方式：GET


 入参说明


| 参数 | 描述 | 类型 | 可否为空 | 备注说明 |
| --- | --- | --- | --- | --- |
| method | 方法名称 | 字符串(String) | 不可为空 | 固定为: updateReviewItem |
| reviewId | 流程id | 字符串(String) | 不可为空 | 流程id |
| itemValueMap | 表单数据 | JSON | 允许为空 |  |
| reviewOperation | 流程操作 | JSON | 允许为空 | 看后面，为空代表只更新数据 |


reviewOperation说明


| 参数 | 描述 | 类型 | 可否为空 | 备注说明 |
| --- | --- | --- | --- | --- |
| handlerLoginName | 用户域账号 | 字符串(String) | 不可为空 | 特权人域账号 |
| operationType | 操作类型 | 字符串(String) | 不允许为空 | 可选值: admin_jump-节点跳转 admin_abandon-流程废弃 handler_pass -流程通过 |
| jumpToNodeId | 要跳转的目标节点编码 | 字符串(String) | 不允许为空 |  |
| auditNote | 备注 | 字符串(String) | 不允许为空 |  |


##### 举例


请求


```

curl --location --globoff 'https://域名/cvte/api/ekpKmReview.do?method=updateReviewItem&reviewId=18d3a86ab8bf567df4e03fd4df198c64&itemValueMap={"fd_tv_licence": "有","fd_3a2f0055c0d20e": [{"LoginName": "liangfenglin"}]}&reviewOperation={"handlerLoginName": "cplm_admin","operationType":"admin_jump","jumpToNodeId": "N3","auditNote": "初始化数据"}
 ''
```


响应


```
{
    "isOk": true,
    "returnMessage": ""
}
```
