# 查询原生流程定制代码


srChangeList替换为要查的关键字


```
SELECT krt.fd_name AS 模板名称,sxt.FD_DISPLAY_JSP, krt.fd_id AS 模板id
FROM SYS_XFORM_TEMPLATE sxt
LEFT JOIN km_review_template krt ON (krt.FD_ID =sxt.FD_MODEL_ID )
WHERE 1=1
AND sxt.fd_display_jsp LIKE '%srChangeList%'
```


原生流程模板:


```
https://xx.xx.xx/km/review/km_review_template/kmReviewTemplate.do?method=view&fdId=替换为流程模板id
```
