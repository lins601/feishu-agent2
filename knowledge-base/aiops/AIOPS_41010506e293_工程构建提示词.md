# 工程构建提示词

# 01 cursor生成项目


# 项目背景


搭建一个IT统一智能运维管理后端，需要你帮我初始化项目工程


# 技术栈要求


依赖管理使用：maven
框架技术：最新版本的spring boot、对应的spring cloud
数据库：使用mybatis
配置中心使用：Apollo
jdk使用：17
api文档使用：swagger
本地缓存使用：caffeine
分布式缓存使用：Redis


## 数据库


获取配置使用：postgresql
数据库连接池：druid


# 工程说明


## 工程总体


是一个多模块分层，总的工程名为ai-ops，细分有ai-ops-common,ai-ops-core,ai-ops-api,ai-ops-starter


## 包路径


1.基础包路径是：com.cvte.ai.ops
2.各模块的基础包路径是：基础包路径+模块特点标识，例如：com.cvte.ai.ops.common


## ai-ops-common


通用代码模块，用于存放常量、枚举、工具类、DTO基类、注解等等
基础包路径为：com.cvte.ai.ops.common


## ai-ops-core


核心业务模块，用于业务实现、数据库(repository、mapper等)操作
基础包路径为：com.cvte.ai.ops.core


## ai-ops-api


rest接口模块，用于对外请求访问和拦截器web相关内容
基础包路径为：com.cvte.ai.ops.api


## ai-ops-starter


启动模块，用于项目启动配置和启动类
基础包路径为：com.cvte.ai.ops.starter
