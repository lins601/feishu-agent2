# 查询系统待办(不带OA和简道云)

## 1 总述


| 栏目 | 说明 | 备注 |
| --- | --- | --- |
| 请求方式 | GET |  |
| 请求数据格式 | param |  |
| 返回数据格式 | JSON |  |
| 地址 | /v1/todo/forward/system/list |  |


## 2 入参说明


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| systemName | String | 是 | 系统来源 |  |
| state | String | 是 | 待办已办未办状态 | 0-已办 1-待办 |
| sid | String | 是 | 单据id |  |


## 3 出参说明RestResponse中data为List


| 参数名 | 数据类型 | 说明 | 备注 |
| --- | --- | --- | --- |
| id | String | 待办id |  |
| sid | String | 单据id |  |
| attribute1 | String | 待办条件表达式 |  |
| handlerUser | String | 审批人域账号 |  |
| handlerName | String | 审批人姓名 |  |


## 4 入参示例


```
https://csbsit-api.gz.cvte.cn/todo-front/v1/todo/forward/system/list?systemName=CPLM&state=1&sid=yyye222a11
```
