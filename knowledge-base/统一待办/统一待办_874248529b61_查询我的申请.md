# 查询我的申请


## 接口关键说明


| 接口关键值域 | 说明 | 备注 |
| --- | --- | --- |
| 接口URL | `https://csbsit-api.gz.cvte.cn/todo-api/v1/todo/teams/unified/process_apply` | 正式环境: `https://csb-api.gz.cvte.cn/todo-api/v1/todo/teams/unified/process_apply` |
| 接口请求方式 | GET |  |
| 交互数据格式 | Form |  |
| 请求数据对象 | x-auth-token（当前用户身份令牌） |  |
| 返回对象 | RestResponse |  |
| 鉴权方式 | x-auth-token 鉴权 |  |


备注：


请求头增加 x-forword-referer=api,否则会报客户端不能为空


---


## 1.1 请求数据说明


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| keyword | String | 否 | 搜索关键词 |  |
| pageNum | Integer | 否 | 页码 | 默认 1 |
| pageSize | Integer | 否 | 每页条数 | 默认 20 |
