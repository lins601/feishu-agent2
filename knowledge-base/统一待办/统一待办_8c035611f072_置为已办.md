## 接口关键说明


| 接口关键值域 | 说明 | 备注 |
| --- | --- | --- |
| 接口URL | `https://csbtest-api.gz.cvte.cn/todo-front/v1/todo/forward/{handler}/done/sid/{sid}` | 正式环境: `https://csb-api.gz.cvte.cn/todo-front/v1/todo/forward/{handler}/done/sid/{sid}` |
| 接口请求方式 | PATCH |  |
| 请求数据对象 | handler: 用户域账号，sid: 单据号 |  |
| 返回对象 | RestResponse |  |
| 鉴权方式 | IAC |  |


---


## 1.1 请求数据说明


| 参数名 | 数据类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| handler | String | 是 | 用户域账号（Path 参数） |
| sid | String | 是 | 单据号（Path 参数） |
