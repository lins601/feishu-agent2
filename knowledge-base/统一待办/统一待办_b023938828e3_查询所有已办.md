# 查询所有已办

[https://kb.cvte.com/pages/viewpage.action?pageId=348795771](https://kb.cvte.com/pages/viewpage.action?pageId=348795771)


# 已办查询接口


## 接口关键说明


| 接口关键值域 | 说明 | 备注 |
| --- | --- | --- |
| 接口URL | `https://csbsit-api.gz.cvte.cn/todo-api/v1/todo/teams/unified/finish` | 正式环境: `https://csb-api.gz.cvte.cn/todo-api/v1/todo/teams/unified/finish` |
| 接口请求方式 | GET |  |
| 交互数据格式 | Form |  |
| 请求数据对象 | handler（登录用户名账号） |  |
| 返回对象 | RestResponse |  |
| 鉴权方式 | JWT 鉴权 |  |
| 客户端标识 | 请求头 x-forword-referer |  |


---


## 1.0 请求头


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| x-auth-token | String | 是 | JWT 或 auth-token |  |
| x-forword-referer | String | 是 | 客户端标识 |  |


---


## 1.1 请求参数说明


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| keyword | String | 否 | 关键字 | 待办标题/内容 |
| sourceKey | String | 否 | 待办来源 | 个人待办：发起人；系统待办：来源系统 |
| pageNum | Integer | 否 | 页码 | 默认 1 |
| pageSize | Integer | 否 | 每页条数 | 默认 20 |


---
