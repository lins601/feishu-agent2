# 注入事件

## 文件位置


```
import type {  IComponentEventRegistryItem, } from '@cvte/mobile-framework';
```


## 字段说明


### 组件事件


```
interface IComponentEventRegistryItem {
  code: string;                    // 组件代码
  group?: string;                  // 事件分组
  batchTranslate?: () => Promise<any>;  // 批量翻译函数
  formatDefaultValue?: () => Promise<any>;  // 格式化默认值函数
  searchBatch?: () => Promise<any>;  // 批量搜索函数
}
```


### 基础事件


```
interface IEventRegistryItem {
  code: string;                    // 事件代码（唯一标识）
  title: string;                   // 事件标题
  description: string;             // 事件描述
  event: () => Promise<any>;       // 事件处理函数
}
```


## 注入示例


### 组件事件


```
// src/components/business/injectEvents.ts
import type { IComponentEventRegistryItem } from '@cvte/mobile-framework';

export const injectEvents: IComponentEventRegistryItem[] = [
  {
    code: 'SELECT',
    group: '__SELECT',
    batchTranslate: () => import('@/components/business/select/batchTranslate'),
    formatDefaultValue: () =>
      import('@/components/business/select/formatDefaultValue'),
  },
];
```


### 基础事件


```
export const baseEventConfig: IEventRegistryItem[] = [
  {
    code: '__myCustomEvent',
    title: '我的自定义事件',
    description: '自定义事件示例',
    event: () => import('@/components/business/forms/utils/events/myCustomEvent'),
  },
];
```


### 服务端事件


```
export const injectServerEvents: IEventRegistryItem[] = [
  loadTreeConfig,
];
```


## 引入和注册


### 客户端事件注册


在 `src/components/layout/MobileFrameworkWrapper.tsx` 中引入并注册：


```
// src/components/layout/MobileFrameworkWrapper.tsx
import { registerComponentsEvents, registerEvent } from '@cvte/mobile-framework';
import {
  baseEventConfig,
  injectEvents,
} from '@/components/business/injectEvents';

// 在文件底部注册组件事件
registerComponentsEvents(injectEvents);
// 注册基础事件
registerEvent(baseEventConfig);
```


### 服务端事件注册


在 `src/app/layout.tsx` 中引入并注册：


```
// src/app/layout.tsx
import { registerComponentsEvents, registerServerEvent } from '@cvte/mobile-framework';
import { injectServerEvents } from '@/components/business/injectEvents';

// 在文件底部注册服务端事件
registerComponentsEvents(injectEvents);
registerServerEvent(injectServerEvents);
```
