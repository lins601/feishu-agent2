# 后端事项

# 1菜单集成


##1.1 CSB legox菜单体系
csb-legox（BSM/CBA/CPLM-BOOT）需要升级csb-sys-base到3.0.4-SNAPSHOT/2.0.9.8-SNAPSHOT用于获取菜单信息，有新增接口


## 1.2 非CSB 菜单体系


需要按照接口格式提供接口，参考接口编写菜单接口即可com.cvte.csb.sys.api.controller.MenuApiController#listAllMenuPages


### 1.2.1 接口路径


目前要求路径必须是这个格式，前缀可以自由决定
**{域名+转发路径等}**/api/v1/menu/page/all


### 1.2.2 接口返回


标准的json结构返回，符合IT开发规范，返回的contentType需要是json


```
{
  "data": [],
  "stauts": "0",
  "message": "success"
}
```


data 返回List <SysMenuDTO>菜单列表
