# 概览

# COS产品运营平台


### 核心特性


**1. 数据采集**


- 基于 Sentry 二次封装，稳定可靠地上报与收集性能数据


**2. 页面性能监控**


- 集成原系统菜单，查阅菜单性能或功能便捷
- 支持主动上报功能性能情况


**3. 运营分析**


- 支持查看系统访问情况
- 支持查看页面访问情况


**4. 进阶功能（二期开发）**


- 🎯 **功能展示**： 展示性能自定义上报
- 📊 **运营分析**：功能级运营数据查看与分析
