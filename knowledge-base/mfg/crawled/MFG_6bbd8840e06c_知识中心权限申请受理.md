# 知识中心权限申请受理

# 业务申请权限
# 处理步骤：

1.让业务联系对应文件所属BG的体系工程师，让文件所属BG的体系工程师评估是否可开通权限，由文件所属BG的体系工程师来开通权限；


- 注意：IT不直接开权限，IT在大多数情况下是评估不了权限开通的合理性；
