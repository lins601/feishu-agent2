# 状态机对接文档


>


**前言**：状态机主要包含两部分的内容：**状态流管理**和**状态流设计器**

cskills add state-flow-engine

## 一、对接方式（以2.2版本接入举例，如果需要2.3的版本，需要替换相应的资源名称和导出点、版本号）


确保系统已经接入资源中心，如果没有接入，可以参考文档：[资源中心接入指南](https://docs.cvte.com/docs/fee/fee-resource-center-usage-integrate-csb)


### 1. 接入状态流管理


例如：`page.tsx`


```
import Loader, { ReactRemoteLoaderComponent } from '@cvte/resource-center-sdk';

// 状态流管理组件
const FlowManager = ReactRemoteLoaderComponent(
  new Loader({
    appName: '', // 系统资源名称
    name: 'cir-state-flow-manager'
  }),
  'FlowManager',
  {
    useShared: true,
    mode: 'page'
  }
);

const page = () => {
  return <FlowManager {...props} jumpTo={(id, code) => {}} />;
};
```


**参数说明：**


| 属性 | 说明 | 类型 |
| --- | --- | --- |
| `useApp` | 是否是多应用，默认填 `false` | `boolean` |
| `apiConfig` | 接口配置，`core`: 状态机接口转发/域名。例如：`https://lcptest.gz.cvte.cn/core` | `object` |
| `jumpTo` | 业务系统跳转到其他模块的逻辑。`(id, code) => void`；`id`：当前点击的状态流模版 id；`code`：当前点击的状态流模版编码。需要把 `id` 和 `code` 传递给下面的 `FlowDesign` 组件，入参分别为 `templateId`、`templateCode` | `Function` |


### 2. 接入状态流设计器


```
import Loader, { ReactRemoteLoaderComponent } from '@cvte/resource-center-sdk';

// 状态流设计器组件
const FlowDesign = ReactRemoteLoaderComponent(
  new Loader({
    appName: '', // 系统资源名称
    name: 'cir-state-flow-manager'
  }),
  'FlowDesign',
  {
    useShared: true,
    mode: 'page'
  }
);

const page = () => {
  return <FlowDesign {...props} />;
};
```


**参数说明：**


| 属性 | 说明 | 类型 |
| --- | --- | --- |
| `useApp` | 是否是多应用，默认填 `false` | `boolean` |
| `apiConfig` | 接口配置。`core`: 状态机接口转发/域名，例如：`https://lcptest.gz.cvte.cn/core`；`csbApi`: csb 的接口转发/域名，例如：`https://csbtest-admin.gz.cvte.cn/legox/admin/v1` | `object` |
| `location` | 内部路由地址，参数必须携带状态流模板 `id` 和状态流模板 `code` | `object` |


### 3. 状态流管理和状态流设计器的集成案例


```
import React, { FC, useState } from 'react';
import { Drawer } from 'antd';
import Loader, { ReactRemoteLoaderComponent } from '@cvte/resource-center-sdk';

import IProps from './index.d';
import './index.less';

// 状态流管理组件
const RemoteFlowManager = ReactRemoteLoaderComponent(
  new Loader({
    appName: '', // 系统资源名称
    name: 'cir-state-flow-manager'，
    version: '' // 指定版本号
  }),
  'FlowManager',
  {
    useShared: true,
    mode: 'page'
  }
);

// 状态流设计器
const RemoteFlowDesign = ReactRemoteLoaderComponent(
  new Loader({
    appName: '', // 系统资源名称
    name: 'cir-state-flow-manager'，
    version: '' // 指定版本号
  }),
  'FlowDesign',
  {
    useShared: true,
    mode: 'page'
  }
);

const FlowManager: FC<IProps> = (props) => {
  const [flowDesignModalVisible, setFlowDesignModalVisible] = useState<boolean>(false);
  const [locationParams, setLocationParams] = useState<{ id: string; code: string }>(null);

  const handleOpenFlowDesign = (id, code) => {
    setFlowDesignModalVisible(true);
    setLocationParams({ id, code });
  };

  return (
    <>
      <RemoteFlowManager
        useApp={false}
        apiConfig={{ core: '/apis/common/proxy/lcpStateFlow' }}
        jumpTo={handleOpenFlowDesign}
      />
      <Drawer
        visible={flowDesignModalVisible}
        title="流程图设计"
        placement="bottom"
        contentWrapperStyle={{ height: 'calc(100% - 45px)' }}
        onClose={() => setFlowDesignModalVisible(false)}
      >
        <RemoteFlowDesign
          useApp={false}
          apiConfig={{
            core: '/apis/common/proxy/lcpStateFlow',
            csbApi: '/apis/common/proxy/lcpCsb'
          }}
          templateId={locationParams?.id}
          templateCode={locationParams?.code}
          location={{ id: locationParams?.id, code: locationParams?.code }}
        />
      </Drawer>
    </>
  );
};

export default FlowManager;
```


>


**说明 1**：`/apis/common/proxy/lcpStateFlow` 和 `/apis/common/proxy/lcpCsb` 是 csb 系统的接口转发配置，还需要在 `config/server.js` 写上对应的转发配置。（`lcpStateFlow`、`lcpCsb` 的字段名可以修改成其他的名字，只是个接口转发，不和其他的转发名称相同即可）


>


**说明 2**：如果接口出现 404，请用 Postman 验证请求的接口是否异常。比如以下接口 404，在 Postman 验证这个接口是否调用异常：`https://csbtest-admin.gz.cvte.cn/legox/admin/v1/role`


### 4. 接入审批日志


```
import Loader, { ReactRemoteLoaderComponent } from '@cvte/resource-center-sdk';

// 审批日志
const FlowLog = ReactRemoteLoaderComponent(
  new Loader({
    appName: '', // 系统资源名称
    name: 'cir-csb-generator'，
    version: '' // 指定版本号
  }),
  'LayoutCsbChangeLog',
  {
    useShared: true,
    mode: 'page'
  }
);

// 流程图
const FlowChart = ReactRemoteLoaderComponent(
  new Loader({
    appName: '', // 系统资源名称
    name: 'cir-csb-generator'
  }),
  'workflowCharts',
  {
    useShared: true,
    mode: 'page'
  }
);

const page = () => {
  return <FlowLog {...props} />;
};
```


**参数说明：**


| 属性 | 说明 | 类型 |
| --- | --- | --- |
| `pageId` | 页面 id | `string` |
| `apiConfig` | api 转发配置。`core`: 状态机接口转发/域名，例如：`https://lcptest.gz.cvte.cn/core` | `{ core: string, path: string }` |
| `changeLogTableExtraConfig` | AntdPaginationTable 的配置，会覆盖组件中已经配置的 AntdPaginationTable config | `object` |
| `flowCharExtraConfig` | 流程图的 modal 和流程图配置 | `{ modalConfig: ModalProps, flowCharHeight: number }` |
