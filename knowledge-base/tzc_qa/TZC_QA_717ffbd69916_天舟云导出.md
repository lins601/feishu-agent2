# 天舟云导出

### 问题现象：


CSB视图开发的列表页点击工具栏上的导出按钮，没有可选字段,怎么排查？


### 排查过程:


- 初步定位：用户没有该视图的导出权限。


### 问题修复：


```
 登录CSB后台，业务角色管理->视图权限->输入列表名称->选择对应系统视图->设置可导出字段
```
