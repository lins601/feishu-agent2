# 创建表单数据 (不再使用)

新接口: [https://docs.cvte.com/docs/tzdoc_v2/tzdoc_v2-1g3vrlaju3lt1](https://docs.cvte.com/docs/tzdoc_v2/tzdoc_v2-1g3vrlaju3lt1)


# 1. 总述


| 栏目 | 说明 | 备注 |
| --- | --- | --- |
| 请求方式 | POST |  |
| 请求数据格式 | JSON |  |
| 返回接口格式 | JSON |  |
| 接口域名 | 查看低代码后端app服务负载均衡配置 |  |
| 接口地址 | /v1/app/form/{classApi} |  |
| 鉴权方式 | IAC/JWT | 建议JWT |


## 1.1 请求头信息


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| x-app-id | String | 是 | 应用ID | 也可以放入路径参数中 |
| x-tenant-id | String | 是 | 租户ID | 也可以放入路径参数中 |


# 2. 入参说明


 路径参数


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| classApi | String | 是 | 表单分类ID或者表单分类编码 |  |


ObjFormDataReq entityDTO


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| dataVersion | DataVersionDTO | 否 | 数据版本 |  |
| allObjAttrItemMap | Map<String, ObjFormDataAttrDTO> | 否 | 表单主表所有字段值 | 用于事件快捷取值 |
| objAttrItemList | List< ObjFormDataAttrDTO> | 是 | 属性清单 |  |
| bizList | Map<String, List<Map<String, Object>>> | 否 | 明细表数据 |  |


## 2.1 ObjFormDataAttrDTO 属性对象


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| attrId | String | 是 | 属性ID |  |
| apiName | String | 否 | 属性API | ID和API二选其一 |
| attrValue | Object | 是 | 属性值 | 真实值 |


## 2.2 Map<String, List<Map<String, Object>>> 明细表数据


| 参数名 | 数据类型 | 是否必填 | 说明 | 备注 |
| --- | --- | --- | --- | --- |
| key | String | 是 | 明细表名 | 即明细表APINAME |
| value | List<Map<String, Object>> | 是 | 所有行数据 |  |
| Map<String, Object> | Map<String, Object> | 是 | 行数据 | 字段API和真实值 |
