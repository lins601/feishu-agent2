## 1 配置说明


- 天舟云只需要配置必要的配置，多个服务公用一个即可，如有项目独有的，可以自定义个namespace配置即可
```yaml

# 数据库配置，必须配置成自己的
spring.datasource.url = jdbc:postgresql://t-dbsec-postgres.gz.cvte.cn:11000/pguat_scm
spring.datasource.username = scm
spring.datasource.password = XXXXXX
spring.datasource.driver-class-name = org.postgresql.Driver
#Redis连接，建议配置成自己的
spring.redis.host = redis-fat001.gz.cvte.cn
spring.redis.password = XXXXX
spring.redis.port = 6379
spring.redis.database = 9

# IAC配置，必须配置成自己的
csb.iac.appId = 15d22276-a397-4f24-bcea-XXXXXX
csb.iac.appSecret = d96deb81-358e-481b-XXXXXX
csb.iac.systemId = 1206b5b9-1c37-45c6-8ed0-XXXXX
csb.iac.tenantId = 93caaf4e-478e-4026-a87d-XXXXXXX

# 服务名，必须配置成自己的
lcp.hub.application.name = scm-lcp-hub
lcp.app.application.name = scm-lcp-app
lcp.data.application.name = scm-lcp-data
csb.legox.application.name = scm-bsm

# 流水号使用公共的，建议使用自己的
csb.serial.feignName = csb-legox/legox

# 队列，使用低代码的MQ使用不同的队列
itemtoes.queue.name = scm-lcpSearchPushItemToEsQueue


#域名配置，域名转发必须一致，都是/hub /app /data，域名改成自己的
lcp.core.url = [https://scm-buat-api.gz.cvte.cn/app](https://scm-buat-api.gz.cvte.cn/app)
lcp.data.url = [https://scm-buat-api.gz.cvte.cn/data](https://scm-buat-api.gz.cvte.cn/data)
lcp.hub.url = [https://scm-buat-api.gz.cvte.cn/hub](https://scm-buat-api.gz.cvte.cn/hub)

# bsm csb legox后台访问地址


service.csb.url = [https://scm-buat-api.gz.cvte.cn](https://scm-buat-api.gz.cvte.cn)
service.csb.gw.url = [https://csbuat-gw.gz.cvte.cn](https://csbuat-gw.gz.cvte.cn)

#流水号地址，如果需要使用自己的需要修改,多租户版本使用[https://csbsit-admin.gz.cvte.cn/legox，单租户使用](https://csbsit-admin.gz.cvte.cn/legox，单租户使用)
service.csno.url = [https://csbsit-admin.gz.cvte.cn/legox](https://csbsit-admin.gz.cvte.cn/legox) 或者 [https://csbsit-api.gz.cvte.cn/csno](https://csbsit-api.gz.cvte.cn/csno)

# 产品线标识，改成自己的


lcp.log.prodLine = scm

```
